Surfin

Statistical Inference for Random Forests

Download .tar.gz View on GitHub

Description

This R package computes uncertainty for random forest predictions using a fast implementation of random forests in C++. Two variance estimates are provided: U-statistics based (Mentch & Hooker, 2016) and infinitesimal jackknife on bootstrap samples (Wager, Hastie, Efron, 2014), the latter as a wrapper to the authors' R code randomForestCI.

Check out a demo: How Uncertain Are Your Random Forest Predictions?

Update

This is an exciting time for research into the theoretical properties of random forests. This R package aims to provide all state-of-the-art variance estimates in one place, to make it easier for random forest practitioners to compare estimates, as well as expedite research in this area.

Variance estimates coming soon! (1) Bag of little bags (Sexton and Laake 2009) (2) Infinitesimal jackknife on subsamples (Wager & Athey, 2017; Athey, Tibshirani, Wager, 2016) as a wrapper to the authors' R package grf.

Feedback, bug reports, know of another variance estimate? Contact us!

Installation

Soon to be uploaded to CRAN. For now, download from this website the .tar.gz (whether you are on Mac, Linux, or Windows). Note that it is a source, not binary file, and needs to be compiled using C++ development tools. If you don't already have the dependencies (Rcpp, RcppArmadillo, Matrix, knitr) and optional dependencies (randomForest, rpart) installed, install those from CRAN first. If you are on Windows, make sure you have RTools installed for C++ development tools. If you already have an older version of surfin installed, remove that first by typing the following in R: 

$ remove.packages("surfin")
Then install using:

$ install.packages(path_to_downloaded_file, repos=NULL, type="source")
$ library(surfin)
See help file at:

$ ?surfin

And check out the demo.
